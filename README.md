## Input examples


```
Parallel: |
Series: +
-------
Three parallel resistors:
1M|1k|100

Three 1k resistors in parallel, which are parallel with 20k:
(1k|1E3|1000)|20k

Two 1k resistors in parallel in series with a 100 Ohm resistor
(1k|1k)+100

Two 1k resistors in series, parallel with a 2k resistor

(1k+1k)|2k

```
